Install

One command. Claude CLI installs Clayrune for you on your local machine.

macOS / Linux
curl -sSL https://clayrune.io/install.sh | sh
Tested on Ubuntu 22.04+, macOS 14+. WSL Ubuntu also works.
Windows (PowerShell)
iwr https://clayrune.io/install.ps1 -useb | iex
Tested on Windows 11. May need Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass first if your policy is restrictive.

What the installer does

  • Installs Claude CLI if missing (npm or Anthropic's installer)
  • Installs Node.js 20 via nvm/winget if you don't have it
  • Clones this repo to ~/Clayrune (or %USERPROFILE%\Clayrune)
  • Sets up the Python venv + dependencies
  • Creates a desktop shortcut + Start Menu / Applications entry
  • Launches the server and opens http://localhost:5199 in your browser

The full install logic is a Markdown prompt — Claude reads it and executes the steps. Read the prompt before you run.
